HOW TO PLAY COLOR MEMORY

A color will flash briefly in the square. Try to memorize it!

Then use the color picker to select the color from memory.

Scoring

  • Exact match → +2 points
  • Close match → +1 point
  • Far off → 0 points

First player to 10 points wins!
